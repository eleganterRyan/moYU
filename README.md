## v0
**关键文件**：fake_word.py
三方库：tkinter PIL cv2 numpy

**其他文件**：
save_background.py：保存软件背景图片
download_icon.py：下载软件图标
resource_path.py：导出exe辅助文件
build_exe.py：打包exe文件


主要功能：
1. 创建类似word界面的txt阅读器。老板的克星，员工的好助手。
2. 打开文件：ctl+o
3. 关闭软件：右上角叉叉
4. 最大化软件：右上角放大按钮
5. 将软件拖动到屏幕最上方自动全屏
6. 移除软件最上角的标题栏
7. 设施软件的名字始终为“文档1”
8. 使用图像处理技术确定文本导入区域
